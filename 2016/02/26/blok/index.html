一个简单的博客

标签（空格分隔）： node express mongodb

#安装Express

npm install -g ex

press-generator – 在全局安装Express

#新建一个工程
在命令行输入：

express -e xxxx
cd xxxx && npm install

#运行

DEBUG=xxxx node ./bin/www

在浏览器里访问 localhost:3000

#文件目录

  1. app.js 启动文件，入口文件
  2. bin: 存放可执行文件
  3. node_modules: 存放各种模块文件
  4. package.json: 工程信息以及依赖模块信息
  5. public: 存放 images js css等文件
  6. routes: 存放路由文件
  7. views: 存放视图或者说模板文件
